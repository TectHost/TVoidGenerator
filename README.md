# TVoidGenerator
Simple world generator

# Features
- Empty World Generator
- World tp
- World importer

# Commands
- /tvoidgenerator reload
- /tvoidgenerator generate
- /tvoidgenerator tp
- /tvoidgenerator import

# Permissions
- tvoidgenerator.reload
- tvoidgenerator.generate
- tvoidgenerator.tp
- tvoidgenerator.import

# Dependencies
- None
